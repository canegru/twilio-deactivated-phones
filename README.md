# twilio-deactivated-phones

# Description

Quick script which gets all deactived phone numbers from Twilio using your account credentials.

### Installing dependencies

```shell
$ npm i
$ npm i ts-node-dev -g
```

### Setup environment file

```shell
$ cat .env.template > .env
```

### Starting

```shell
$ npm start
```

### Tests

```shell
$ npm run test
```

### Building

```shell
$ npm run build
```
